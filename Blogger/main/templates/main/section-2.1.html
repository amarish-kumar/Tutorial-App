Open terminal(Ctrl+Alt+T), go to folder where project will live then type `dotnet new web -n <Name Of Your Project>` without angle brackets (In my case project is called Algodemy), when installation finishes open project folder with your prefer editor in my case is vscode.

This should be project structure you see

and use terminal that is already open from project parent folder and type `cd <Name Of Your Project >` to change current directory then another command `dotnet run` to test whether application can run properly.

If everything passed dotnet will up local server and listen in port 5000

navigate to localhost:5000 and you probably see this Hello World! message and without errors in dev-tools console.
