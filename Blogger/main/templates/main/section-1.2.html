For Angular Command Line Tools is just simple as open terminal (Ctrl+Alt+T) and paste `npm install -g @angular/cli` this should install Angular CLI tools globally in your machine(few seconds/minutes). When installation finish check this as type `ng --version` then you already see similarly thing respective to version you install if everything is went well so far, if not always is good idea to refer to vendors documentation Angular CLI repo

And here we are ready for learn and prepare development environment with two of the latest and modern server and client side technology.
