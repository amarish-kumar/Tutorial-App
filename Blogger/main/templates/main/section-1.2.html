For Angular Command Line Tools is just simple as open terminal (Ctrl+Alt+T) and paste ` npm install -g @angular/cli` this should install Angular CLI tools globally in your machine(few seconds/minutes). When installation finish check this as type in terminal ` ng --version` then you should see similarly thing respect to the version you install.

And here we are ready for learn and prepare development environment with two of the latest and modern server and client side technology.
