Open terminal in .NET Project folder(not client app) and type `npm install` this command should install all Angular dependencies.

Here in "node_modules" folder with all angular dependencies and npm packages live

Type on last opened console `ng build` little bit later in “wwwroot” folder appear new one with name “ClientApp” just as we configuring CLI before, there is all compiled Typescript in form of JavaScript files that we need for our purpose along with source maps which is used for debugging but in this tutorial is not relevant for us.

See "wwwroot" folder after `ng build` command with all Javascript

All chunks as we see are the same as in root folder.

Right now we have all the parts in place, and remains to test whether everything works smoothly but before that we must do a little more work such as configuring pipline middleware, creating a controller to manage the coming requests and return apropriate Razor page with right scripts injected in it.
